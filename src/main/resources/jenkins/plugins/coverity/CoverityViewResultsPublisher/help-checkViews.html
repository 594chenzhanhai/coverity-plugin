Checks the Coverity Connect Instance, Project, and View to ensure valid configuration. This checks the following:
  • The Coverity Connect Instance value is a valid configured global instance
  • The Coverity Connect Instance user has permissions to fetch defects
  • The Project specified exists on that Coverity Connect Instance
  • The View is configured in that Coverity Connect Instance
    • Will provide warnings in the case where the view is missing required information
