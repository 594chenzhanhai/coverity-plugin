Choose your source control system (SCM) to enable revision history to be retrieved, both for display in the Coverity Connect UI and to influence test policies. Some SCMs require configuration

NOTE: Some SCMs are not supported in every supported version of the Coverity Static Analysis Tools. These are:

  • Team Foundation Server 2017 is only supported in versions 2017.07 and newer
  • Plastic is only supported in versions 2017.07 and newer
  • Plastic (fully distributed) is only supported in versions 2017.07 and newer
